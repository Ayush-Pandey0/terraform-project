# Terraform AWS EC2 Project

## Overview
This project provisions an AWS EC2 instance using Terraform.

## Specifications
- **Cloud Provider:** AWS
- **Region:** ap-south-1
- **OS:** Amazon Linux (AMI: ami-0f58b397bc5c1f2e8)
- **Instance Type:** t2.micro
- **Tags:** Name = Terraform-Student-Instance

## Setup & Deployment Steps

1. Initialize the Terraform working directory:
   ```bash
   terraform init
   ```

2. Preview the changes that will be made:
   ```bash
   terraform plan
   ```

3. Apply the changes to create the resources:
   ```bash
   terraform apply
   ```

## Submission
This project includes all required Terraform files and this README containing the setup steps.
